# How to use this?

- Ensure you already clone backend and website.

The file struct belike:

```
your_folder_name/
├── backend/
├── docker/
│   ├── docker-develop-componse.yml       
│   └── nginx/                   
└── website/
```

And cd to docker typing:

For develop:
```bash
docker compose -p instructhub_dev -f ./docker-develop-compose.yml up -d
```
